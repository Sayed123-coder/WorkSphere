# WorkSphere – Professional Networking Platform

This project is a full-stack implementation of a professional networking platform inspired by LinkedIn.
It replicates core social and professional interaction features such as user profiles, connections, posts, and feed management while focusing on scalable architecture, state management, and real-world backend design.

The goal of this project was not just UI replication, but building a production-style system with proper authentication, data flow, and modular architecture.

## 🚀 Features
🔐 JWT Authentication (Login / Register)
👤 Dynamic User Profiles with work & education details
📝 Create, delete, and view posts
🤝 Send, accept, and manage connection requests
📰 Personalized feed based on connections
💬 Comment system on posts
🔎 View other user profiles
📦 Global state management using Redux Toolkit
⚡ Optimized API calls and async data handling
🎯 Responsive UI
